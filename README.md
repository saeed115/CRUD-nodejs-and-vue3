# protrans

api app with node and express

# Server

### after CD into Server folder

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm start
```

# Client

### after CD into Client folder

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
