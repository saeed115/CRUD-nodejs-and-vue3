# protrans

api app with node and express

# back-end

### after CD into back-end folder

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run start
```

# front-end

### after CD into front-end folder

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
